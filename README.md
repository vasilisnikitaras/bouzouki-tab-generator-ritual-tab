# 🎼 Τελετουργική Ταμπλατούρα για Τετράχορδο Μπουζούκι

Ένα εργαλείο για μετατροπή νοτών, συχνοτήτων και τραγουδιών (YouTube ή Suno) σε ταμπλατούρα για τετράχορδο μπουζούκι. Με διάρκεια, γράφημα, και AI προτάσεις για ελάχιστη μετακίνηση. Φτιαγμένο με Streamlit για βαθιά, τελετουργική μάθηση.

---

## 🔧 Λειτουργίες

- 🎵 Είσοδος με νότα ή συχνότητα
- 🎧 Ανάλυση αρχείων ήχου (`.mp3`, `.wav`) και YouTube links
- 🧠 Αυτόματη αναγνώριση νοτών και διάρκειας
- 🎼 Ταμπλατούρα με χορδή, τάστο και διάρκεια
- 🔮 AI προτάσεις για εναλλακτικές θέσεις με λιγότερη μετακίνηση
- 📊 Οπτικοποίηση σε γράφημα
- 📤 Εξαγωγή σε εικόνα ή PDF (προσεχώς)

---

## 🚀 Εκτέλεση τοπικά

```bash
pip install -r requirements.txt
streamlit run app.py
